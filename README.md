hiimzein

Minimal personal landing page with interactive cards and a reactive, layered hero background.

Features
- Custom cursor with hover growth
- Scroll reveal animations for cards
- 3D tilt interaction on cards
- Layered hero background (grid, starfield, aurora, glow)
- Tunable background reactivity to mouse and scroll
- Respects prefers-reduced-motion and disables mouse effects on touch devices

Usage
- Static site: open index.html in a browser.

Files
- index.html — markup
- styles.css — styles and background layers
- script.js — interactions, parallax, cursor, canvas comets
